# warfarin-pgx

**A pharmacogenomics-based algorithm for personalized warfarin dose prediction.**

Warfarin is a widely used anticoagulant with a narrow therapeutic window, and the right dose varies significantly from patient to patient. This project explores a **pharmacogenomic (PGx) approach** to estimating a personalized warfarin dose by combining clinical factors with genetic information.

## What's inside

| File | Description |
|------|-------------|
| `Warfarin_dose_Alg_V1.py` | First version of the dosing algorithm. |
| `Warfarin_4_Algorithm_Comparator.xlsx` | Spreadsheet comparing dosing algorithm outputs. |
| `warfarin_patients.db` | Sample patient dataset used for development and testing. |

## Goals

- Estimate individualized warfarin dosing using clinical and pharmacogenomic inputs.
- Compare multiple dosing algorithms to evaluate accuracy and consistency.
- Build a foundation for precision-medicine tooling in anticoagulation therapy.
